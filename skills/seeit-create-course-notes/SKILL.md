# SeeIt 课程笔记

使用 SeeIt AI MCP 工具检索课程证据，并输出可复习、可回看的结构化笔记。

## 工作流

1. 确定 `media_id`。未提供时调用 `list_media`；只有一个候选时直接使用，多个候选时请求用户确认。
2. 调用 `get_video_metadata`，确认文件名、来源、状态和证据数量。
3. 分别调用 `search_video_evidence` 检索：
   - 课程主题、章节和核心概念；
   - 定义、原理、公式或关键结论；
   - 示例、对比、易错点和总结。
4. 对不同时间段的高相关命中调用 `get_evidence_window`，补齐前后语境。避免对同一时间戳重复展开。
5. 证据过少或检索退化到时间轴开头时，调用 `get_video_timeline` 扫描相应范围；仍不足时明确说明限制。
6. 用 `verify_video_citations` 批量校验最终引用。删除不受支持的内容，或重新检索后修订。

## 输出要求

按以下顺序输出：课程概览、章节笔记、核心概念、示例或易错点、复习清单、复习题。每个事实性结论至少附一个 `[MM:SS]` 时间戳；保留 ASR/OCR 来源。

只根据工具返回的证据写作。不要补写视频未明确讲述的定义、答案或背景知识。证据无法确定时使用“视频中未明确说明”。

## 分析任务

用户要求保存完整报告时，调用 `start_video_analysis`，使用具体目标，例如“生成课程章节、概念解释、易错点和带时间戳的复习笔记”。用 `get_analysis_status` 查询任务，并在完成后用 `get_analysis_report` 读取结果。不要高频轮询；任务未完成时返回任务 ID 和当前状态。
